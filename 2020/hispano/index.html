October 12, 2020

JakartaOne in Spanish is a one-day virtual conference in Spanish for developers and technical leaders bringing the current state and future of Jakarta EE and related technologies, focused on the enterprise cloud-native application development.

The event has now concluded!

Thank You for making the first JakartaOne Livestream in Spanish event a great success!

This milestone would not be possible without the collective efforts of a dedicated community.

Talks Helidon. A dual option for developing microservices in the cloud View Helidon. A dual option for developing microservices in the cloud abstract

Helidon is a framework for building microservices, supported by Oracle. Besides being attached to Microprofile (3.2) it also allows the creation of reactive code. In this talk we will see all the benefits it offers to be an option for the development of microservices for the cloud.

 		Isaac Ruiz Session video for session Helidon. A dual option for developing microservices in the cloud
