12 de octubre del 2020

JakartaOne en Español es una conferencia virtual para desarrolladores y líderes técnicos que ofrece el estado actual y el futuro de Jakarta EE y tecnologías relacionadas, enfocado en el desarrollo de aplicaciones empresariales nativas para la nube.

El evento ha finalizado!

Gracias a todos quienes hicieron que el primer JakartaOne Livestream en Español event a great success !

Esta meta no hubiera sido posible sin el esfuerzo colectivo de nuestras comunidades.

Charlas Kubernetes Nativo con Eclipse MicroProfile Ver Kubernetes Nativo con Eclipse MicroProfile abstract

Kubernetes es la plataforma de-facto platform para desplegar nuestras aplicaciones hoy en día. Pero este movimiento también ha introducido algunos desafíos. En esta sesión veremos cuales son estos desafíos y demostramos cómo resolverlos con Eclipse MicroProfile.

Charlas JakarteEE in Action Ver JakarteEE in Action abstract

Desarrollar una aplicación empresarial básica utilizando varias de las especificaciones de JakartaEE como Jakarta Bean Validation, Jakarta Persistence, Jakarta Enterprise Bean, Jakarta Server Faces y Jakarta Security que permite entender tanto el ecosistema de Jakarta como la forma de integrarlos.

 		Marco Toscano Video de sesión para sesión JakarteEE in Action
Charlas Crear tests que permiten flow Ver Crear tests que permiten flow abstract

Por la mayoría de los proyectos empresariales, tests no son realmente agradables. Es aburrido, incómodo, y lleva su tiempo, de que crear tests. De todos modos, tests son importantes; entonces, como podemos abordarlos esos en una manera efectiva?

 		Sebastian Daschner Video de sesión para sesión Crear tests que permiten flow
Charlas Pavimentando el Camino con Jakarta EE 9 y Apache TomEE 9.0.0 Ver Pavimentando el Camino con Jakarta EE 9 y Apache TomEE 9.0.0 abstract

La migración del paquete javax a jakarta en Jakarta EE 9 introduce un reto en la plataforma, el ecosistema y por ende en usuarios finales.En esta sesión se detallan las recomendaciones y estrategias que pueden ayudar a la migración de tus aplicaciones con el proyecto de código abierto Apache TomEE.

 		Cesar Hernandez Video de sesión para sesión Pavimentando el Camino con Jakarta EE 9 y Apache TomEE 9.0.0
Charlas Escalando aplicaciones Jakarta EE de manera vertical y horizontal con Jelastic PaaS Ver Escalando aplicaciones Jakarta EE de manera vertical y horizontal con Jelastic PaaS abstract

Analicemos cómo mantener las aplicaciones de Jakarta EE elásticas y eficientes, compartamos los secretos de tunning y el seguimiento de la carga, así de cómo automatizar el escalado vertical y horizontal.

 		Fabio de Carvalho Video de sesión para sesión Escalando aplicaciones Jakarta EE de manera vertical y horizontal con Jelastic PaaS

Charlas Jakarta EE Tips- Filtros e interceptores Ver Jakarta EE Tips- Filtros e interceptores abstract

¿Cansado de repetir tus validaciones y repetir tu código en cada endpoint ? ¿Aburrido de tener que llenar los campos de auditoría de tus entidades a mano? ¡No se diga más! Aprende como extraer la lógica común de tus aplicaciones con Filtros, Listeners e interceptores!

 		Jorge Cajas Video de sesión para sesión Jakarta EE Tips- Filtros e interceptores
Charlas Lo bueno de JSF con lo nuevo de JakartaEE Ver Lo bueno de JSF con lo nuevo de JakartaEE abstract

A lo largo de todos estos años, los Java Server Faces han sido el framework predilecto por muchos debido a su robustez y seguridad para construir aplicaciones web basadas en Java. A pesar de sus desventajas ampliamente discutidas sigue siendo una muy buena alternativa.

 		Jorge Vargas Video de sesión para sesión Lo bueno de JSF con lo nuevo de JakartaEE
Charlas Sigue tu aplicación en la nube en tiempo real con MicroProfile Metrics! Ver Sigue tu aplicación en la nube en tiempo real con MicroProfile Metrics! abstract

Una de las principales preocupaciones que existen en el dessarollo de aplicaciones modernas en la nube es el monitoreo eficaz de estas en tiempo real. Esta charla mostrara como preparar una aplicacion empresarial para monitoreo en tiempo real mediante el uso de Eclipse MicroProfile Metrics!

 		Fabio Turizo Video de sesión para sesión Sigue tu aplicación en la nube en tiempo real con MicroProfile Metrics!
Charlas Helidon. Una opción dual para desarrollar microservicios en la nube Ver Helidon. Una opción dual para desarrollar microservicios en la nube abstract

Helidon es un framework para la construcción de microservicios, apoyado por Oracle. Ademas de estar apegado a Microprofile (3.2) permite también la creación de código reactivo. En esta charla veremos todas las bondades que ofrece para ser una opción para el desarrollo de microservicios para la nube.

 		Isaac Ruiz Video de sesión para sesión Helidon. Una opción dual para desarrollar microservicios en la nube
Charlas Jakarta EE Security Specification Ver Jakarta EE Security Specification abstract

Exploraremos los requerimientos de seguridad que determina el estándar de Jakarta EE deben llenarse por parte de los productos desarrollados y cumplir con características de seguridad tales como autenticaciones, control de acceso a recursos, integridad de la data, entre otros.

 		Mercedes Wyss Video de sesión para sesión Jakarta EE Security Specification
